## 摘要
零样本目标导航要求具身智能体在没有先前训练的情况下探索并定位目标物体。为此，近期方法利用了基础模型，但它们通常依赖静态先验知识且缺乏适应性，导致重复错误和昂贵的试错成本。本文提出了一种自进化的零样本目标导航框架，能够在测试时持续改进。具体来说，我们通过从过往轨迹中提取可行动知识来构建智能体规则记忆。然后，我们提出了一种基于置信上界的检索策略，通过平衡语义相关性和历史成功来选择有效规则。此外，我们引入了一个记忆引导的预反思模块，在行动前预测潜在结果，减少低效探索。大量实验表明，我们的方法优于现有的零样本基线，实现了10.1%的成功率提升，同时减少了不必要的步骤。

## Abstract
Zero-Shot Object-Goal Navigation (ZS-OGN) requires embodied agents to explore and locate target objects without any prior training. To this end, recent methods leverage foundation models. But they typically rely on static priors and lack adaptation, which leads to repeated errors and costly trial and error. In this paper, we propose a self-evolving ZS-OGN framework that enables continuous test-time improvement. Specifically, we build an agentic rule memory by extracting actionable knowledge from past trajectories. Then, we propose a retrieval strategy based on upper confidence bound, selecting effective rules by balancing semantic relevance and historical success. In addition, we introduce a memory-guided preflection module that forecasts potential outcomes before action, reducing inefficient exploration. Extensive experiments show that our method outperforms existing zero-shot baselines, achieving a 10.1\% improvement in success rate with fewer unnecessary steps.

## 1. 核心问题与整体含义（研究动机）
- 现有零样本物体目标导航方法依赖静态先验知识（如固定常识），缺乏在线适应性，导致智能体在不同环境中重复相同错误（例如被视觉相似物体误导），且试错成本高昂。
- 在有限的步数预算下，每一步行动都很昂贵。现有方法往往在动作执行后才修正错误（后验校正），浪费大量步骤。
- 因此，论文核心挑战是：如何在有限测试步数预算下，同时实现在线适应和行动预判，从而减少不必要的物理试错。

## 2. 方法论
- **核心思想**：将导航转化为连续自改进过程，通过提取历史轨迹中的可行动导航规则，并利用“预反思”在行动前预测风险，实现零样本场景下的训练无关、自进化导航。
- **关键技术细节**：
  - **自进化智能体规则记忆（Self-evolving Agentic Rule Memory）**：在每个导航片段结束后，通过LLM分析轨迹，提取关键步骤并通过语义相似度（房间类型+可见物体与目标类别的余弦相似度）计算每个步骤的信用分数（Credit = Si - S_{i-1}），再经softmax归一化，得到新生成规则的支持分数 Sr。利用动量更新规则效用 μ_r，最终采用上置信界（UCB）公式：UCB(r) = μ_r + β √(ln T / N_r) 平衡语义相关性与历史成功率；新规则初始UCB设为无穷大以保证探索。
  - **记忆引导的预反思模块（Memory-Guided Preflection）**：在每次决策时，系统先为每个候选前沿点（Frontier）检索top-K规则，然后要求LLM结合规则与当前局部观察进行风险预测（例如判断该前沿是否会导致死路、目标不可达等），最后选择风险最低、期望收益最高的方向。
- **算法流程**：
  1. 感知阶段：获取RGB-D图像、位姿，构建2D占据网格地图并提取候选前沿。
  2. 预反思决策：从规则记忆库中检索top-K规则，LLM对每个前沿进行风险推理，选定最优目标点。
  3. 低层控制器执行移动（最多H步，若到达目标则触发STOP）。
  4. 片段结束后：LLM分析轨迹，提取关键步骤并生成新规则；根据步骤信用更新规则效用μ_r和UCB分数，插入规则库。
