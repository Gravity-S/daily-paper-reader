## 摘要
为了提升强化学习在现实世界中的适用性，对抗性鲁棒强化学习领域研究如何在对抗环境扰动下训练智能体。在此设置中，主角智能体在对抗者的环境扰动下优化策略，形成零和马尔可夫博弈。当对抗性鲁棒强化学习与基于模型的强化学习结合时，对抗者可以针对学习到的转移模型而非训练环境。基于这一思路，本工作引入了深度强化学习智能体在推理时的后验鲁棒化。通过学习到的模型与训练好的名义策略相结合，我们的方法执行鲁棒策略改进步骤，旨在无需额外训练神经网络的情况下提升鲁棒性。具体而言，我们在对抗性展开下利用模型预测控制，通过在有界不确定性集内的投影梯度下降进行近似。此外，这些离线展开在执行时考虑并缓解了分布外问题。通过在受扰动的Gymnasium MuJoCo环境中评估算法，同时考虑后验推理设置的计算限制，验证了所提方法在鲁棒性上的显著提升。

## Abstract
To improve the real-world applicability of reinforcement learning (RL), the field of adversarially robust RL studies how to train agents under adversarial environment perturbations. In this setting, a protagonist agent optimizes a policy under environmental perturbations from an adversary, resulting in a zero-sum Markov game. When adversarially robust RL is combined with model-based RL, the adversary can target a learned transition model instead of the training environment. Extending this idea, this work introduces post-hoc robustification of deep RL agents at inference time. By using the learned model in combination with a trained nominal policy, our approach performs a robust policy improvement step. The goal is to improve robustness without any additional training of neural networks. Specifically, we utilize model-predictive control under adversarial rollouts, which are approximated via projected gradient descent within a bounded uncertainty set. Furthermore, these offline rollouts are performed while considering and mitigating out-of-distribution issues. The proposed methodology is validated by demonstrating significant improvements in robustness when the algorithm is evaluated in perturbed Gymnasium MuJoCo environments, while considering the computational limitations of the post-hoc inference setting.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：标准强化学习（RL）智能体在训练环境变化（如仿真到现实、仿真到仿真）时性能严重下降，而现有的对抗鲁棒RL方法通常需要在训练阶段引入对抗扰动，导致训练不稳定、性能下降或需要大量额外计算。本文旨在**在不修改训练过程的前提下，在推理阶段提升已预训练智能体的鲁棒性**，即“后验鲁棒化”（post-hoc robustness）。
- **背景**：基于模型的RL（MBRL）通过学习环境转移模型来模拟交互。本文利用该模型，结合训练好的名义策略（nominal policy），在推理时通过模型预测控制（MPC）进行鲁棒策略改进，无需重新训练神经网络。目标是在计算受限的推理环境下，显著提升智能体对物理参数扰动（如质量、摩擦、阻尼等）的鲁棒性。

## 2. 论文提出的方法论

### 核心思想
- 在推理时，每步使用学习到的转移模型进行**悲观滚动**（pessimistic rollouts），即由对抗者扰动预测的下一个状态，使其最小化价值函数，然后由策略优化（外问题）选择使累积回报最大的动作。方法类似于鲁棒MDP的零和博弈，但使用名义价值函数（而非鲁棒价值函数）作为近似。

### 关键技术细节
1. **不确定性集定义**：采用最优传输成本（Optimal Transport Cost, OTC）不确定性集，以L2范数定义状态扰动半径 \(r_2\)，使扰动后的状态 \(\mathbf{s}' = \bar{\mathbf{s}}' + \delta\) 满足 \(\|\delta\|_2 \leq r_2\)。
2. **对抗滚动生成**：每次模型预测时，通过**投影梯度下降（PGD）** 迭代更新扰动 \(\delta\)，以最小化下一状态的价值函数 \(V_{\psi}(\mathbf{s}')\)：
   \[
   \delta \leftarrow \text{Proj}_{\|\delta\|_2 \leq r_2}\left( \delta - \alpha \frac{\nabla_{(\mathbf{s}'+\delta)} V_{\psi}(\mathbf{s}'+\delta)}{\|\nabla_{(\mathbf{s}'+\delta)} V_{\psi}(\mathbf{s}'+\delta)\|_2} \right)
   \]
   通常1-3次迭代即可收敛到局部最优。
3. **外问题求解**：使用模型预测路径积分（MPPI）算法优化局部策略 \(\pi_s^{\text{MPC}}\)，使其最大化对抗滚动下的累积回报。
4. **缓解分布外问题（OOD）**：采用集成模型（ensemble）的差异作为不确定性度量，当自展误差超过阈值 \(h\) 时截断滚动深度，避免模型在不确定区域被过度利用。该方法来自Kidambi等人（MOReL）。

### 算法流程（文字说明）
- 预训练阶段：用MBPO训练名义策略 \(\pi_{\psi}\) 和模型集成 \(p_{\theta}\)。
- 推理阶段（每步）：
  - 使用MPPI算法，在模型上生成N步对抗滚动（每步状态经PGD扰动）。
  - 对每个候选动作序列，计算悲观累积回报（最后一状态用名义价值函数 \(V_{\psi}\) 近似）。
  - 选择使悲观回报最大的动作执行。
  - 若滚动过程中模型不确定性超过阈值，则提前终止滚动。

## 6. 论文的主要结论与发现

1. **后验鲁棒化显著提升鲁棒性**：在所有扰动场景下，鲁棒MPC的回报衰减远小于基线MBPO和非鲁棒MPC。
2. **对抗扰动是关键**：非鲁棒MPC在Hopper-v4上几乎无改善，在Reacher-v4上部分改善，但结合对抗扰动后才达到最佳效果。
3. **OOD缓解必不可少**：不采用截断阈值时，模型被过度利用，鲁棒性甚至低于基线（图2）。
4. **滚动深度和半径的敏感性**：增加滚动深度N和扩大扰动半径r2通常能提升鲁棒性，但过大会导致模型误差积累或性能下降，存在稳定工作区间（如Hopper-v4中r=0.045~0.075，N=40）。
5. **计算开销可接受**：在高端GPU上每步0.09~0.15秒（Hopper-v4）或0.015~0.024秒（Reacher-v4），适合推理场景。

## 7. 优点

- **无需额外训练**：直接在推理阶段提升鲁棒性，与训练过程解耦，避免对抗训练带来的训练不稳定和性能损失。
- **即插即用**：可应用于任何已训练的MBRL智能体，只需少量超参数调整（不确定性半径、滚动深度、阈值）。
- **方法透明**：基于PGD的对抗滚动和模型集成的OOD检测有直观理论基础，且消融实验清楚地分离各组件贡献。
- **计算效率高**：通过Jax实现并利用GPU加速，推理延迟在可接受范围内。
