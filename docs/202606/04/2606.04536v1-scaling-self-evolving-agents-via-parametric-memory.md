## 摘要
现有的记忆增强型LLM智能体仅以文本摘要或检索段落的形式在提示空间中存储过往经验，同时在整个执行过程中保持模型参数冻结。这类智能体可以“查找”其所见内容，但无法从中“学习”：其策略不会因经验而改变，任何从上下文中丢弃的信息都会永久丢失。我们引入TMEM，一种自演化的参数化记忆框架，其中智能体不仅将历史压缩成显式记忆，还通过轻量级在线更新将提炼得到的监督信息吸收到快速LoRA权重Δ_t中，从而在单个情节内真正改变其未来行为。我们将其形式化为具有快速权重执行动态的智能体决策过程：动作从π_{θ_0+Δ_t}中采样，同时提取动作生成监督信息，用于更新后续决策的Δ_t。这一观点使得提取策略可直接通过强化学习进行优化：训练θ_0不仅改进行动质量，还改进了用于在线LoRA适配的数据质量。我们进一步提出基于SVD的LoRA子空间初始化以加速在线收敛。在LoCoMo、LongMemEval-S、多目标搜索和CL-Bench上的实验表明，TMEM在不同模型规模上始终优于基于摘要和基于检索的基线方法。

## Abstract
Existing memory-augmented LLM agents store past experience exclusively in prompt space, as textual summaries or retrieved passages, while keeping model parameters frozen throughout a rollout. Such agents can \emph{look up} what they have seen but cannot \emph{learn from} it: their policy is unchanged by experience, and any information dropped from the context is permanently lost. We introduce \texttt{TMEM}, a self-evolving parametric memory framework in which the agent not only compresses history into explicit memory but also absorbs distilled supervision into fast LoRA weights $Δ_t$ via lightweight online updates, genuinely altering its future behavior within a single episode. We formalize this as an agentic decision process with fast-weight rollout dynamics: actions are sampled from $π_{θ_0+Δ_t}$, while extraction actions produce supervision that updates $Δ_t$ for subsequent decisions. This view makes the extraction policy directly optimizable by RL: training $θ_0$ improves not only task actions but also the quality of the data used for online LoRA adaptation. We further propose SVD-based initialization of the LoRA subspace to accelerate online convergence. Experiments on LoCoMo, LongMemEval-S, multi-objective search, and CL-Bench show that \texttt{TMEM} consistently outperforms summary-based and retrieval-based baselines across different model scales.

- **问题**：现有记忆增强型 LLM 智能体仅在提示空间中存储过去的经验（如文本摘要或检索段落），在完整执行过程中模型参数保持冻结。智能体只能“查找”历史，无法真正“从经验中学习”——策略不随经验改变，任何从上下文丢失的信息都永久消失。
- **动机**：长期任务（多轮对话、长序列搜索等）中，智能体需要超越单个上下文窗口的经验；理想状况下记忆应当改变未来的决策方式，而非仅仅作为额外文本注入。
- **贡献**：提出 TMEM，一种自演化的参数化记忆框架，使智能体能在单集内通过在线轻量级 LoRA 更新将提炼的监督信息写入快速权重（fast weights），从而实质性改变后续行为。

### 2. 方法论
- **核心思想**：将智能体执行过程形式化为一个快速权重展开（fast-weight rollout）过程：策略 π 依赖于工作上下文 h_t、显式记忆 m_t 以及快速参数化记忆 Δ_t（用 LoRA 权重表示）。当上下文预算达到时，智能体进入记忆写入模式，将当前会话蒸馏为基于 QA 的监督，并通过在线 SFT 更新 Δ_t。后续动作从自适应策略 π_{θ₀+Δ_t} 采样，其中基参数 θ₀ 在集内固定。
- **关键技术细节**：
  - **在线 LoRA 更新**：仅对 Transformer 中最后 4 层的 FFN 投影（gate_proj, up_proj, down_proj）附加 LoRA 适配器，秩 r=6；只在在线更新阶段训练系数矩阵 B，投影矩阵 A 固定。
  - **SVD 初始化**：对每个目标权重 W = UΣV^T，取前 r 个右奇异向量和奇异值：B₀=0, A₀=Σ_r V_r^T。B₀=0 保证初始策略与基模型一致，但 A₀ 已经覆盖了 W 的主能量方向，加速在线少步适应。
  - **停止梯度策略优化**：使用强化学习（GRPO）优化基参数 θ₀，目标函数为期望结果奖励 J(θ₀)。梯度通过每个片段（segment）的普通动作和边界提取动作的反向传播计算，但通过 Δ_t 和在线更新算子 T 的路径应用 stop-gradient。这确保了 RL 信号同时改进任务行为和提取质量。
- **数学形式化**：
  - 普通动作：a_t ∼ π_{θ₀+Δ_t}(·|q, h_t, m_t)
  - 提取动作（触发时）：a_{t_i} ∼ π_{θ₀+Δ_{t_i}}(·|q, h_{t_i}, m_{t_i}, d)，其中 d 为记忆写入提示，要求生成结构化 QA 监督。
  - 更新方式：TMEM 中 (m_{t_i+1}, Δ_{t_i+1}, h_{t_i+1}) = (∅, T(Δ_{t_i}, a_{t_i}), ∅)；摘要基线中更新 Δ_t 不变，显式记忆携带 a_{t_i}。

### 7. 优点
- **方法创新**：将参数化记忆（快速权重）与智能体决策过程统一在一个 rollout 框架内，形式上统一了现有显式记忆方法（Δ_t ≡ 0 时退化为纯提示空间记忆）。
- **理论支撑**：定理 1 提供了 SVD 初始化相对于随机初始化近似误差更小的理论保证（在谱对齐假设下）。
- **实验全面**：覆盖了多种长程任务（对话、搜索、持续学习），且在前 RL 和后 RL 两个阶段评估，消融实验全面，统计报告规范。
- **效率可控**：虽然在线 LoRA 更新引入了额外开销，但 TMEM 比纯检索更快，比无记忆基线更省内存；通过调节触发预算 L_max 可以平衡精度与成本。
- **实用性**：方法不需要大幅修改基础模型或依赖外部知识库，可以直接应用于现有 LLM。

  - CL-Bench 过滤子集基于 Qwen3-30A3B 的能力筛选，可能引入选择偏差，导致结果不能直接推广到原始完整分布。

- **计算资源报告不完整**：未提供 GPU 型号、总训练时间、能源消耗等，影响可重复性。

- **单集内学习限制**：TMEM 只在单集内更新 Δ_t，跨集的知识积累还需要额外机制（如外层元学习）。
