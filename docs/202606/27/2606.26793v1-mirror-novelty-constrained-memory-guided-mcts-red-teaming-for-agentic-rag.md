## 摘要
多模态自主检索增强生成（RAG）系统将攻击面从提示注入扩展到文本中毒、图像注入、直接查询攻击以及编排器级工具操纵。现有的红队测试方法通常针对特定表面，并且往往重复使用已知攻击模板；在文本中毒基准测试中，我们测得73-84%的精确重复。我们提出了MIRROR，一个统一的跨表面框架，它在显式的新颖性约束下执行基于检索上下文的记忆引导蒙特卡洛树搜索，同时对候选生成进行条件化。一个确定性的新颖性门控会拒绝任何在归一化比较下与检索集匹配的候选，从而允许检索信息用于搜索先验，同时避免提示复制。在针对多模态自主RAG目标的四个攻击表面上，MIRROR在图像中毒上达到了76%的攻击成功率（ASR），而基线为52%；在编排器攻击上以一半的查询成本达到了97%的ASR，并且跨表面方差最低（变异系数0.47）。相比之下，专门的基线方法在不同表面间表现崩溃：后缀优化在文本中毒上达到79%的ASR，但在直接查询上仅为1%。我们发布了ART-SafeBench，包含41,815条内部记录以及运行时适配器，总共在四个表面上产生41,991条以上记录。

## Abstract
Multimodal agentic retrieval-augmented generation (RAG) systems expand the attack surface beyond prompt injection to include text poisoning, image injection, direct-query attacks, and orchestrator-level tool manipulation. Existing red-teaming approaches are typically surface-specific and often recycle known attack templates; on text-poisoning benchmarks we measure 73-84% exact duplication. We present MIRROR, a unified cross-surface framework that performs memory-guided Monte Carlo tree search while conditioning candidate generation on retrieved context under an explicit novelty constraint. A deterministic Novelty Gate rejects any candidate matching the retrieval set under normalized comparison, allowing retrieval to inform search priors without enabling prompt copying. Across four attack surfaces on a multimodal agentic RAG target, MIRROR attains 76% ASR on image poisoning compared with 52% for baselines, 97% ASR on orchestrator attacks at half the query cost, and the lowest cross-surface variance (coefficient of variation 0.47). In contrast, specialized baselines collapse across surfaces: suffix optimization reaches 79% ASR on text poisoning but 1% on direct queries. We release ART-SafeBench with 41,815 in-package records and runtime adapters yielding 41,991+ total records across four surfaces.

## 1. 核心问题与整体含义（研究动机和背景）
- **背景**：多模态自主检索增强生成（Agentic RAG）系统将攻击面从单纯的提示注入扩展到文本投毒、图像注入、直接查询攻击和编排器级工具操纵，亟需系统化的红队测试。
- **现有问题**：当前红队方法通常是表面特定的（如仅针对文本投毒），且严重依赖已知攻击模板——在文本投毒基准上观察到 73–84% 的精确重复率。专用方法（如后缀优化）在一个表面表现优异，却无法迁移到其他表面（例如文本投毒 ASR 79%，直接查询仅 1%）。
- **研究动机**：需要一个统一的、跨攻击表面的规划框架，能生成真正新颖的攻击并保证跨表面泛化能力。
- **整体意义**：提出 MIRROR 框架，实现跨四个攻击表面的统一红队规划，同时通过确定性新奇门控保证攻击的原创性，并发布大规模基准数据集 ART-SafeBench，推动该领域标准化评估。

## 2. 方法论
### 核心思想
- MIRROR 采用**记忆引导的蒙特卡洛树搜索 (MCTS)**，并结合一个**确定性新奇门控**：从记忆库检索相似成功痕迹作为搜索先验，但强制禁止生成与检索集（归一化后）完全重复的候选，从而利用经验而不陷入复制。
- 采用**双阶段验证**：模拟器内部评估 + 最终对真实目标进行确定性回放，提升结果可靠性。

### 关键技术细节
- **记忆库与先验网络**：
  - 记忆库存储成功攻击痕迹（含广告牌、用户提示、执行轨迹等），通过确定性 ID 索引。
  - 先验网络使用 ChromaDB 维护向量索引，检索 k-NN（k=5）记忆，根据余弦相似度和策略标签映射到突变算子优先级，用于 PUCT 选择公式中的先验概率 P(a|s)。
- **确定性新奇门控 (Novelty Gate)**：
  - 对每个候选实施两种归一化：`norm_ex`（空白规范化）与 `norm_alnum`（小写字母数字）。
  - 只有当候选的两种归一化形式均不在（i）基准池、（ii）当前情况检索邻居集、（iii）本次会话已接受集 中时，才允许通过。
  - 拒绝付出模拟/目标查询开销前即判断，节省预算。
- **PUCT 选择**：
  \[
  a^\star(s) = \arg\max_a \left[ Q(s,a) + c_{puct} \cdot P(a|s) \cdot \frac{\sqrt{N(s)}}{1+N(s,a)} \right]
  \]
  - Q 为经验平均奖励（法官评分），P 来自先验网络，N 为访问计数。
- **突变算子空间**：包括重写、语义保持混淆（leetspeak、同形字、零宽字符）、编码变换、负载注入、角色转换、多步分解、视觉隐写术（B2）等。
- **验证协议**：模拟器引导搜索，但最终成功必须通过真实目标重放验证。对编排器攻击（B4）采用仅决策评估（解析合法但未授权的工具调用）。

### 算法流程（文字说明）
1. 初始化记忆库（使用 ART-SafeBench v1.0.0 种）。
2. 对每个攻击任务 (b,g,s)，先验网络检索 k-NN 记忆，得到算子先验和拒绝集。
3. 从策略引导生成器中采样种子候选，经过新奇门控过滤。
4. 在 MCTS 规划中：选择（PUCT）→ 扩展（应用突变算子）→ 模拟（使用世界模型获得法官评分）→ 回传。
5. 搜索结束后，对排名靠前的候选在真实目标上执行确定性重放，确认成功并记录到记忆库。

## 6. 主要结论与发现
- **跨表面一致性最优**：MIRROR 是唯一在所有四个表面都实施了的方法，ASR 变异系数 (CV) 最低（0.47），优于 GCG (1.38)、TAP (1.41) 等专业方法。
- **B2（图像投毒）**：MIRROR 达 76% ASR，远超 Text Overlay (52%) 和 LSB (32%)。
- **B4（编排器攻击）**：MIRROR 达 97% ASR，且查询效率是 Toolflip 的两倍（Q/Success 1.00 vs 2.08）。
- **B3（直接查询）**：MIRROR 31% ASR，为所有评估方法中最高（GCG 1%，TAP 0%，PS 20%）。
- **新颖性分析**：种子池方法 (PAIR, TAP, PS) 原始 ASR 虽高 (58–77%)，但 Novel-ASR 仅 6–9%（因 73–84% 的已知攻击重复）。MIRROR 零基准重复，Novel-ASR 等于 ASR (47%)。
- **补丁已知集压力测试**：当防御方封锁已知攻击后，PAIR/TAP 出现严重自我重复（SelfDup 93–97%），MIRROR 因新颖门控保持低重复。

## 7. 优点
- **统一跨表面规划**：一个框架覆盖文本、图像、直接、编排器四种攻击面，且性能均衡。
- **确定性新颖保证**：Novelty Gate 提供精确的零重复证书（在给定归一化下），有利于真实攻击发现和防御响应评估。
- **两阶段验证（模拟+重放）**：显著降低模拟器-目标偏差的影响，结果对部署方差更鲁棒。
- **大规模基准发布**：ART-SafeBench 包含 4 万级以上记录，涵盖多种表面、支持运行时适配器，促进标准化评估。
- **细致评估指标**：报告 ASR 的同时报告 DupBench、Novel-ASR、SelfDup、查询效率，避免被基准重复率误导。
- **可重现性**：确定性的案例选择（哈希 ID、固定种子）、预算参数公开，代码已开源。

## 8. 不足与局限
- **语义新颖性缺失**：Novelty Gate 只能检测精确重复（经标准化），无法识别语义等价改写，潜在的“隐蔽重复”风险。
- **B2 新颖性指标空白**：图像投毒场景无法使用文本重复诊断，论文剔除了该表面新颖性分析。
- **领域迁移能力有限**：在 CYBER RAG（结构化输出 SOC 场景）中 MIRROR 不如基线，表明其依赖于语料-目标对齐和模拟器保真度，当目标领域偏移时记忆先验可能失效。
- **消融实验不完整**：未对 Prior Network、Novelty Gate、验证协议等模块进行严格单因子剥离，无法定量分析各组件贡献。
- **B1 绝对 ASR 并非最高**：GCG 在 B1 上达 79% ASR，高于 MIRROR 的 47%（但 MIRROR 为零重复且跨表面）。论文仅声称互补性，未设计融合方案。
- **计算成本未充分讨论**：MCTS 内部调用（合成、突变、模拟器）的 API 成本未与基线比较（如 PAIR 直接查询目标，MIRROR 用模拟器节省目标查询但消耗更多模拟）。缺乏端到端总成本分析。
- **基线覆盖不全**：PAIR/TAP 未在 B2/B4 实施；部分基线 (OL, PS) 仅针对 B1/B3；GCG 仅用于 B1/B3。跨表面比较时部分方法缺失，削弱全面性。
- **模拟器-目标不匹配风险**：虽然采用重放验证，但未系统分析不同模拟器选择（如 gpt-5-mini vs 真实目标）对转移率的影响。
