## 摘要
世界模型为长视界LLM智能体提供了一种原则性的方法：在执行之前预测行动后果。然而，不可靠的预测可能被忽略、误用，甚至降低下游决策质量。本文介绍WorldEvolver，一个自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修正其部署时的上下文。WorldEvolver集成了三个模块：(i) 情景记忆，通过基于检索的模拟利用真实行动转换；(ii) 语义记忆，从预测-观察不匹配中提取持久启发式规则；(iii) 选择性预见，在将预测融入智能体推理上下文之前过滤低置信度预测。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个基线上实现了最高的预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修正提高了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 1. 核心问题与整体含义（研究动机和背景）

论文聚焦于 **LLM 智能体在长程规划中使用世界模型（world model）时的预测可靠性问题**。世界模型被视为提供“预见”（foresight）的关键工具：在执行动作前预测其后果。然而，现实部署中，环境分布会发生偏移（分布漂移），导致冻结（权重不变）的世界模型预测不准确。若将不可靠的预测直接喂给智能体，可能被忽略、误用，甚至降低决策质量（如图2所示，加入噪声预测反而不如无预测）。现有工作要么完全冻结世界模型，要么通过参数更新（如梯度更新）来适应，但后者在大语言模型（LLM）规模下计算成本高，且引发灾难性遗忘或过度编辑风险。

**核心含义**：需要一种 **self-evolving（自我进化）** 的世界模型——在部署过程中在线检测预测与观察的差异，并自适应调整，而不改变模型参数。论文提出将修正方式从“更新模型权重”转变为“修正部署上下文（prompt-level evidence）”，通过维护外部记忆来持续改进预测。

## 2. 方法论

### 核心思想
WorldEvolver 是一个 **冻结下游智能体和所有模型参数** 的框架，通过三个互补的上下文记忆模块实现自我进化：
- **Episodic Memory（情景记忆）**：存储已执行的真实动作转换（`(observation, action, next_observation)`）。对新候选动作，基于动作的 Jaccard 相似度检索 top-k 相似转换，作为检索示例辅助世界模型预测。
- **Semantic Memory（语义记忆）**：将预测-观察的不匹配转化为持久性的启发式规则。使用因子化状态空间（三元组）比较预测与真实观察，当不匹配时，由 LLM critic 提取“规则”（如“检查物体不改变其位置”）。每个规则附带证据分数，只有正分数的规则被保留并注入世界模型的提示上下文。
- **Selective Foresight（选择性预见）**：对世界模型生成的下一个预测，计算其基于 token 几何平均概率的置信度 `q_t`。只有当 `q_t ≥ τ`（阈值）时，才将预测暴露给智能体；否则返回空（相当于无预见）。这避免了低质量预测误导智能体。

### 算法流程（Algorithm 1）
1. 智能体先采样一个草稿动作 `a_t^(0)`。
2. 从情景记忆中检索与该动作相似的 `k_ME` 个转换。
3. 用当前内存（情景+语义）查询冻结世界模型 `W_θ`，得到预测 `ô_{t+1}` 和置信度 `q_t`。
4. 选择性预见：若 `q_t ≥ τ` 则传递预测，否则无预见。
5. 智能体基于状态和（可能的）预见采样最终动作 `a_t`。
6. 若最终动作与草稿动作不同，重新用最终动作查询世界模型以获取对齐的预测。
7. 执行动作，观察真实 `o_{t+1}`。
8. 将真实转换加入情景记忆。
9. 若预测与真实不匹配（因子化后），则进入语义记忆更新（LLM critic 提取规则），可累积 mini-batch（k_MS）后批量更新。

**关键设计**：所有更新都是对非参数记忆的追加或规则提炼，不涉及模型权重更新。

## 6. 主要结论与发现

1. **WorldEvolver 在预测准确率上显著优于所有基线**：在三个骨干、两个环境上，完整 WorldEvolver 的 Exact Match 最高（如 ALFWorld / Gemma-4-26B 达 52.88%，而 RAWM-ϕ 仅 20.06%）。
2. **情景记忆是主导因素**：仅用情景记忆（w/o MS）已超越 RAWM-ϕ，因为它基于动作直接检索转换而非冗长状态-动作文本。
3. **语义记忆带来额外提升**：尤其在 ScienceWorld 上，结合规则后预测更准。
4. **选择性预见提升下游成功率**：添加 Ft 在所有 Gemma-4-26B 设置下均超越无预见基线，且平均超越无世界模型基线 +2.24（ALFWorld）和 +3.33（ScienceWorld）。
5. **弱骨干受益更大**：GPT-5.4-mini 本身更强，改进有限；而 Gemma-4-26B 的规划改善更显著，说明世界模型在次优智能体上补充价值更高。
6. **超参数敏感性**：增大 k_ME（1→5）显著提升预测；k_MS 影响较小。
7. **在线持续学习有效**：在 ScienceWorld 上，随着 trial 数增加，WorldEvolver 的累计成功率提升更明显，表明记忆积累随时间生效。

## 8. 不足与局限

- **环境覆盖有限**：仅在两个文本交互环境（家务、科学实验）上评估，未涉及网页导航、代码生成、机器人控制或多模态交互。这些领域的状态空间可能更复杂，记忆检索和规则提取的实用性有待验证。
- **置信度估计依赖 token 概率**：`q_t` 基于 softmax 概率，对于封闭模型（如 GPT API）可能无法获取，需替代方案（如 self-consistency 或外部校准模型）。此外，置信度与准确度之间的相关性可能因环境而异（文中 Figure 11 显示正相关，但仅在部分设置下验证）。
- **选择性预见阈值 τ 需手动选择**：实验中每个设置单独校准（表3），缺乏自适应机制。不同环境或任务可能需要不同阈值。
- **语义规则提取依赖 LLM critic**：若 LLM critic（与骨干相同）自身偏见或噪声，可能生成无效规则；证据分数更新简单，未考虑规则冲突或长尾规则。
- **记忆管理**：情景记忆无限增长可能超出上下文窗口；论文未讨论记忆修剪或聚合策略。
- **下游提升幅度仍有局限**：对于强智能体（如 GPT-5.4-mini），WorldEvolver 的提升较小甚至有时略低于无世界模型基线（如 ReflAct/ALFWorld 上 -2.99）。说明世界模型并非万能，需与智能体能力相匹配。
