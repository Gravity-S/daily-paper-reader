## 摘要
标准实用拜占庭容错（PBFT）协议专为稳定的有线环境设计，在移动自组网等场景中部署时表现出关键漏洞，容易受到女巫攻击、拜占庭合谋和消息操纵等复杂威胁。现有的静态防御机制不足以应对这些攻击的智能化和协调性。为解决这一挑战，本文利用Agentic AI范式构建分布式多智能体系统，每个共识节点配备一个智能体。这些智能体采用分层多智能体深度Q网络（MADQN）算法，实时学习和执行主动安全策略。通过观察局部网络行为、消息一致性和动态维护的信誉分数，智能体在标准PBFT仲裁和成员规则下协作识别可疑行为并推荐防御措施，从而改善共识过程的完整性。我们将该框架称为Agentic-SecPBFT。然后，正式建模关键攻击向量并进行广泛仿真。结果表明，Agentic-SecPBFT在1.8%误报率下达到95.0%的攻击检测率。与主流PBFT变体相比，在33%恶意节点下，其吞吐量平均提高3.1倍，延迟降低56%，为去中心化无线系统提供了鲁棒且自适应的安全解决方案。

## Abstract
The standard Practical Byzantine Fault Tolerance (PBFT) protocol, designed for stable, wired environments, exhibits critical vulnerabilities when deployed in settings like mobile ad-hoc networks, thus making it susceptible to sophisticated threats such as Sybil attacks, Byzantine collusion, and message manipulation. Existing static defense mechanisms are ill-equipped to handle the intelligent and coordinated nature of these attacks. To address this challenge, this paper leverages the Agentic AI paradigm to build a distributed multi-agent system in which each consensus node is equipped with an intelligent agent. These agents employ a hierarchical Multi-Agent Deep Q-Network (MADQN) algorithm to learn and execute proactive security policies in real-time. By observing local network behavior, message consistency, and dynamically maintained reputation scores, the agents collaboratively identify suspicious behavior and recommend defensive actions under standard PBFT quorum and membership rules, thereby improving the integrity of the consensus process. We refer to the resulting framework as Agentic-SecPBFT. Then, we formally model key attack vectors and conduct extensive simulations. The results demonstrate that Agentic-SecPBFT reaches a 95.0% attack detection rate with a 1.8% false positive rate. Compared with mainstream PBFT variants, it achieves 3.1* higher throughput with 56% lower latency on average under 33% malicious nodes, offering a robust and adaptive security solution for decentralized wireless systems.

- **核心问题**：标准实用拜占庭容错（PBFT）共识协议在设计时假设节点位于稳定、高带宽、可靠的有线网络中。当直接应用于移动自组织网络（如VANETs，无人机群等）时，会面临严峻挑战：
    - **无线环境不可靠**：无线链路的路径损耗、多径衰落（如瑞利/莱斯衰落）会导致频繁的丢包和超时，这与PBFT协议中“超时即视为恶意”的预设相冲突，从而触发昂贵且不必要的视图更换（view-change）过程，严重破坏共识性能和网络活性。
    - **智能协同攻击**：攻击者不再采用静态、易被检测的恶意行为，而是利用无线环境的不确定性作为掩护，发动诸如选择性丢包（On-Off攻击）、女巫攻击（Sybil Attack，即一个物理节点伪造多个逻辑身份）等复杂攻击。这些攻击可以绕过静态的、基于阈值的声誉模型和防御机制。
    - **语义鸿沟**：PBFT的安全模型基于“一个公钥即一个节点”的假设。而在无线网络中，攻击者可以控制一个物理设备，生成多个加密身份，从而破坏PBFT的`n ≥ 3f + 1`（节点数需大于等于3倍故障节点数加1）的安全假设，而诚实节点对此一无所知。
- **整体含义**：该研究旨在解决在去中心化无线系统中，PBFT共识协议的安全性和鲁棒性不足的根本性挑战。其核心是弥合“逻辑身份”与“物理实体”之间的“语义鸿沟”，将被动、静态的防御转变为主动、自适应的安全机制，以应对智能化的网络攻击。

#### 2. 方法论：核心思想与关键技术

- **核心思想**：利用Agentic AI范式，将每个共识节点转变为**自主防御者**。通过构建一个分布式多智能体系统，让每个节点上的“智能体”（Local Consensus Agent, LCA）学习并执行主动安全策略，协作识别并防御针对共识过程的攻击。
- **关键技术细节**：
    - **分层多智能体架构**：
        - **局部共识智能体（LCA）**：部署在每个物理节点上，负责**实时、局部**的威胁感知和响应。它能感知本地信道状态（如SINR、CSI）、共识协议状态（如投票行为、视图号）以及动态维护的声誉。
        - **集群优化智能体（COA）**：在更高层级运行（可部署在基站或通过安全多方计算分布实现），负责**全局、长期**的策略协调。它聚合来自所有LCA的遥测数据，训练全局策略网络，并分发更新后的模型和全局声誉评分。COA的输出是指导性的，不能直接提交区块或修改仲裁阈值，确保了PBFT核心安全规则不会被智能体绕过。
    - **基于分层的多智能体深度Q网络（Hierarchical MADQN）算法**：
        - **问题建模**：将安全问题建模为一个**去中心化部分可观测马尔可夫决策过程（Dec-POMDP）**。每个LCA只能观察到局部环境（如自己的邻居、信道质量、消息等）。
        - **状态空间**：设计了一个包含三层特征的状态向量：
            - **共识特征**：当前PBFT阶段、视图号、邻居投票行为历史、投票信息熵。
            - **网络特征**：邻居节点的SINR向量、CSI空间相关性、消息到达时间方差、干扰强度。
            - **声誉特征**：本地声誉分数、声誉评分随时间变化的方差。
        - **动作空间**：LCA可以执行一系列离散动作，包括：正常参与、标记可疑节点、本地降低其声誉、发起挑战-响应、投票更换视图、临时拒绝其非核心流量、调整传输功率、请求优先时隙。
        - **奖励函数**：一个多目标函数，同时奖励**共识成功**（正确提交区块）、**安全检测**（正确识别攻击）和**网络活性**（避免不必要的视图更换，维持链路质量）。
        - **执行模式**：采用CTDE模式，即**集中训练（COA）、分布执行（LCA）**。训练时，COA聚合经验，用MSE损失更新全局Q网络；执行时，每个LCA独立使用最新的策略进行推理和行动。
    - **动态信道感知的声誉管理**：
        - 与传统模型不同，系统在更新声誉时，会引入一个**物理确定性因子（Ψ_phy）**，该因子结合了链路成功投递概率（P_success）和节点移动性。
        - 当发生丢包或超时时，如果此时SINR很低（信道质量差），则惩罚被减弱；如果SINR很高（信道质量好），则惩罚被加强。这有效地区分了“良性的信道衰落”和“恶意的消息丢包”，显著降低了误报率。
    - **算法流程**：LCA在每个时间步：①感知并预处理物理层和协议层数据；②构建状态向量，通过LSTM处理历史信息形成隐状态；③利用ϵ-贪婪策略选择防御动作；④在PBFT安全护栏内执行动作（如标记、挑战）；⑤评估结果并计算奖励，将经验存储；⑥定期通过Agentic交互协议（AIP）与COA同步，上传经验并下载更新后的模型和声誉。
